## ⛳️ 스터디 진행 방식

---

- 매주 수요일까지 각자 주제의 질문 외 추가로 물어볼 공통 질문을 올립니다.
- 매주 금요일날 해당 주차에 대한 Topic과 함께 공통 질문들을 질문합니다.
- 면접관 2명, 피면접자 1명을 대상으로 돌아가면서 스터디를 진행한 후, Topic과 공통 질문 외에 추가로 나온 질문에 대해서 노션에 기록합니다.
